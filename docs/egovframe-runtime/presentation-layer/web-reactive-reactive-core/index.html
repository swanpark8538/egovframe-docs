ServerCodecConfigurer л№ҲмңјлЎң нҷңм„ұнҷ”н•  мҲҳ мһҲлӢӨ. мҠӨнҠёлҰ¬л°Қ л°©мӢқмңјлЎң multipart лҚ°мқҙн„°лҘј нҢҢмӢұн•ҳл Өл©ҙ HttpMessageReaderк°Җ лҰ¬н„ҙн•ҳлҠ” FluxлҘј мӮ¬мҡ©н•ңлӢӨ. мҳҲлҘј л“Өм–ҙ м»ЁнҠёлЎӨлҹ¬м—җм„ң @RequestPartлҘј м„ м–ён•ҳл©ҙ MapмІҳлҹј мқҙлҰ„мңјлЎң к°Ғ нҢҢнҠём—җ м ‘к·јн•ҳкІ лӢӨлҠ” лң»мқҙлҜҖлЎң multipart лҚ°мқҙн„°лҘј н•ңлІҲм—җ нҢҢмӢұн•ҙм•ј н•ңлӢӨ. л°ҳлҢҖлЎң Flux нғҖмһ…м—җ @RequestPartлҘј мӮ¬мҡ©н—Ҳл©ҙ м»Ён…җмё лҘј л””мҪ”л”©н•  л•Ң MultiValueMapм—җ мҲҳм§‘н•ҳм§Җ м•ҠлҠ”лӢӨ.

Forwarded Headers

Load BalancersмҷҖ к°ҷмқҙ н”„нҶЎмӢңлҘј кІҪмң н•ң мҡ”мІӯмқҖ нҳёмҠӨнҠё, нҸ¬нҠё, URL мҠӨнӮҙмқҙ ліҖкІҪлҗ  мҲҳ мһҲм–ҙм„ң нҒҙлқјмқҙм–ёнҠё мһ…мһҘм—җм„ңлҠ” мӣҗлһҳмқҳ URL м •ліҙлҘј м•Ңм•„лӮҙкё° м–ҙл өлӢӨ. RFC 7239 м •мқҳм—җ л”°лҘҙл©ҙ Forwarded HTTP н—ӨлҚ”лҠ” н”„лЎқмӢңк°Җ мӣҗлһҳ мҡ”мІӯм—җ лҢҖн•ң м •ліҙлҘј м¶”к°Җн•ҳлҠ” н—ӨлҚ”мқҙлӢӨ. X-Forwarded-Host, X-Forwarded-Port, X-Forwarded-Proto, X-Forwarded-Ssl, and X-Forwarded-Prefix к°ҷмқҖ л№„н‘ңмӨҖ н—ӨлҚ”лҸ„ мЎҙмһ¬н•ңлӢӨ. ForwardedHeaderTransformerлҠ” forwarded н—ӨлҚ”лҘј ліҙкі  мҡ”мІӯмқҳ нҳёмҠӨнҠё, нҸ¬нҠё, мҠӨнӮҙмқ„ л°”кҝ”мӨҖ лӢӨмқҢ, н—ӨлҚ”лҘј м ңкұ°н•ҳлҠ” м»ҙнҸ¬л„ҢнҠёмқҙлӢӨ. ForwardedHeaderTransformerлқјлҠ” мқҙлҰ„мңјлЎң л№Ҳмқ„ м •мқҳн•ҳл©ҙ мһҗлҸҷмңјлЎң мІҙмқём—җ м¶”к°ҖлҗңлӢӨ. forwarded н—ӨлҚ”лҠ” ліҙм•Ҳм—җ мӢ кІҪмҚЁм•ј н•  мҡ”мҶҢк°Җ мһҲлҠ”лҚ° н”„лЎқмӢңк°Җ н—ӨлҚ”лҘј м¶”к°Җн•ңкұҙм§Җ, нҒҙлқјмқҙм–ёнҠёк°Җ м•…мқҳм ҒмңјлЎң м¶”к°Җн•ң кІғмқём§Җ м• н”ҢлҰ¬мјҖмқҙм…ҳм—җм„ңлҠ” м•Ң мҲҳ м—Ҷкё° л•Ңл¬ёмқҙлӢӨ. л”°лқјм„ң мҷёл¶Җм—җм„ң л“Өм–ҙмҳӨлҠ” мӢ лў°н•  мҲҳ м—ҶлҠ” н”„лЎқмӢң мҡ”мІӯмқ„ м ңкұ°н•  л•Ң ForwardedHeaderTransformerлҘј removeOnly=trueлЎң м„Өм •н•ҳм—¬ н—ӨлҚ” м •ліҙлҘј мӮ¬мҡ©н•ҳм§Җ м•Ҡкі  м ңкұ°н•  мҲҳ мһҲлӢӨ.

5.1 лІ„м „л¶Җн„° ForwardedHeaderFilterлҠ” м ңкұ°лҢҖмғҒ(deprecated)мқҙ лҗҳм–ҙ ForwardedHeaderTransformerлҘј лҢҖмӢ н•ңлӢӨ. л”°лқјм„ң exchange(http мҡ”мІӯ/мқ‘лӢөкіј м„ём…ҳ м •ліҙ л“ұмқҳ м»Ён…Ңмқҙл„Ҳ)лҘј л§Ңл“Өкё° м „м—җ forwarded н—ӨлҚ”лҘј мІҳлҰ¬н•  мҲҳ мһҲлӢӨ. н•„н„°лҘј мӮ¬мҡ©н•ҳлҚ”лқјлҸ„ мқҙ н•„н„°лҠ” м „мІҙ н•„н„° лҰ¬мҠӨнҠём—җм„ң м ңмҷёлҗҳл©° к·ё лҢҖмӢ  ForwardedHeaderTransformerлҘј мӮ¬мҡ©н•ңлӢӨ.

Filters

WebHandler APIм—җм„ңлҠ” WebFilterлҘј мӮ¬мҡ©н•ҳл©ҙ лӢӨлҘё н•„н„° мІҙмқёкіј WebHandler м „нӣ„м—җ мҡ”мІӯмқ„ к°ҖлЎңмұ„м„ң мӣҗн•ҳлҠ” лЎңм§Ғмқ„ л„Јмқ„ мҲҳ мһҲлӢӨ. WebFilterлҘј л“ұлЎқн•ҳл Өл©ҙ мҠӨн”„л§Ғ л№ҲмңјлЎң л§Ңл“Өм–ҙ л№Ҳ мң„м—җ @OrderлҘј м„ м–ём•„кұ°лӮҳ OrderedлҘј кө¬нҳ„н•ҙ мҲңм„ңлҘј м •н•ҙлҸ„ лҗҳкі , WebFlux ConfigлҘј мӮ¬мҡ©н•ҙлҸ„ лҗңлӢӨ.

CORS

CORSлҠ” м»ЁнҠёлЎӨлҹ¬м—җ м–ҙл…ён…Ңмқҙм…ҳмқ„ м„ м–ён•ҳлҠ” кІғмңјлЎң мһҳ лҸҷмһ‘н•ңлӢӨ. Spring SecurityмҷҖ мӮ¬мҡ©н•ҳл©ҙ CorsFilterлҘј мӮ¬мҡ©н•ҙм„ң Spring Security н•„н„° мІҙмқёліҙлӢӨ лЁјм Җ мІҳлҰ¬н•ҳлҸ„лЎқ н•ҙм•ј н•ңлӢӨ.
