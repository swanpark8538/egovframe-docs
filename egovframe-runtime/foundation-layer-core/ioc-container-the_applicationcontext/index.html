The ApplicationContext

к°ңмҡ”

org.springframework.context нҢЁнӮӨм§ҖлҠ” BeanFactory мқён„°нҺҳмқҙмҠӨлҘј нҷ•мһҘн•ҳлҠ” ApplicationContext мқён„°нҺҳмқҙмҠӨлҘј м¶”к°Җн•ҳкі , лӢӨлҘё мқён„°нҺҳмқҙмҠӨлҘј нҷ•мһҘн•ҳм—¬ ліҙлӢӨ м• н”ҢлҰ¬мјҖмқҙм…ҳ н”„л Ҳмһ„мӣҢнҒ¬ м§Җн–Ҙм Ғмқё мҠӨнғҖмқјлЎң м¶”к°Җ кё°лҠҘмқ„ м ңкіөн•ңлӢӨ.
л§ҺмқҖ мӮ¬лһҢл“Өмқҙ ApplicationContextлҘј мҷ„м „нһҲ м„ м–ём Ғмқё л°©мӢқмңјлЎң мӮ¬мҡ©н•ҳл©°, н”„лЎңк·ёлһҳл°Қ л°©мӢқмңјлЎң мғқм„ұн•ҳм§Җ м•Ҡкі  ContextLoaderмҷҖ к°ҷмқҖ м§Җмӣҗ нҒҙлһҳмҠӨм—җ мқҳмЎҙн•ҳм—¬ Java EE мӣ№ м• н”ҢлҰ¬мјҖмқҙм…ҳмқҳ м •мғҒм Ғмқё мӢңмһ‘ н”„лЎңм„ёмҠӨмқҳ мқјл¶ҖлЎң ApplicationContextлҘј мһҗлҸҷмңјлЎң мқёмҠӨн„ҙмҠӨнҷ”н•ңлӢӨ.

м„ӨлӘ…

ApplicationContextлҠ” BeanFactoryлҘј нҷ•мһҘн•ң кІғмңјлЎң BeanFactoryмқҳ кё°лҠҘ мҷём—җ м•„лһҳмҷҖ к°ҷмқҖ кё°лҠҘмқ„ м ңкіөн•ңлӢӨ.

  • MessageSource : i18n-sytleлЎң л©”мӢңм§ҖлҘј accessн•  мҲҳ мһҲлҸ„лЎқ м§Җмӣҗн•ңлӢӨ.
  • Access to resources : URL, File л“ұкіј к°ҷмқҖ мһҗмӣҗмқ„ мүҪкІҢ accessн•  мҲҳ мһҲлҸ„лЎқ м§Җмӣҗн•ҳлӢӨ.
  • Event propagation : ApplicationListener interfaceлҘј кө¬нҳ„н•ң beanм—җкІҢ EventлҘј м „лӢ¬н•ңлӢӨ.
  • Loading of multiple (hierarchical) contexts : кі„мёө кө¬мЎ°мқҳ contextлҘј м§Җмӣҗн•ЁмңјлЎңмҚЁ, м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқҳ мӣ№ л Ҳмқҙм–ҙ л“ұкіј к°ҷмқҖ нҠ№м • л Ҳмқҙм–ҙм—җл§Ң м§‘мӨ‘м Ғмқё contextлҘј мһ‘м„ұн•  мҲҳ мһҲлӢӨ.

Event

ApplicationContextлҠ” Event мІҳлҰ¬лҘј мң„н•ҙ ApplicationEvent, ApplicationListener interfaceлҘј м ңкіөн•ңлӢӨ. ApplicationListener interfaceлҘј кө¬нҳ„н•ң beanмқҖ ApplicationContextм—җ л°ңмғқн•ң лӘЁл“  ApplicationEventлҘј м „лӢ¬л°ӣлҠ”лӢӨ. Springмқҙ м ңкіөн•ҳлҠ” н‘ңмӨҖ eventлҠ” лӢӨмқҢкіј к°ҷлӢӨ.

Eventм„ӨлӘ…
ContextRefreshedEventApplicationContextк°Җ мҙҲкё°нҷ”лҗңкұ°лӮҳ refreshлҗ  л•Ң л°ңмғқн•ңлӢӨ(refreshн•ҳкё° мң„н•ҙм„ң ConfigurableApplicationContext interfaceмқҳ refresh() л©”мҶҢл“ңлҘј мӮ¬мҡ©н•ңлӢӨ). вҖңмҙҲкё°нҷ”вҖқлқјлҠ” лӢЁм–ҙлҠ”, лӘЁл“  beanмқҙ loadлҗҳм—Ҳкі , post-processor beanмқҙ нғҗм§Җлҗҳм–ҙ нҷңм„ұнҷ”лҗҳм—Ҳмңјл©°, singleton к°қмІҙк°Җ м„ к°қмІҙнҷ”лҗҳм–ҙ, ApplicationContext к°қмІҙк°Җ мӮ¬мҡ©к°ҖлҠҘн•ң мғҒнғңм—җ мһҲлӢӨлҠ” кІғмқ„ мқҳлҜён•ңлӢӨ. refreshлҠ” contextк°Җ лӢ«нҳҖм§Җм§Җ м•ҠмқҖ н•ң, м—¬лҹ¬лІҲ л°ңмғқн•  мҲҳ мһҲмңјл©°, ApplicationContextк°Җ вҖңhotвҖқ refreshлҘј м§Җмӣҗн•ҙм•јн•ңлӢӨ (XmlWebApplicationContextлҠ” вҖңhotвҖқ refreshлҘј м§Җмӣҗн•ҳм§Җл§Ң, GenericApplicationContextлҠ” м§Җмӣҗн•ҳм§Җ м•ҠлҠ”лӢӨ).
ContextStartedEventApplicationContextк°Җ мӢңмһ‘лҗ  л•Ң л°ңмғқн•ңлӢӨ(мӢңмһ‘н•ҳкё° мң„н•ҙм„ң ConfigurableApplicationContext interfaceмқҳ start() л©”мҶҢл“ңлҘј мӮ¬мҡ©н•ңлӢӨ). вҖңмӢңмһ‘лҗЁ(Started)вҖңмқҙлһҖ лӢЁм–ҙлҠ”, лӘЁл“  Lifecycle beanмқҙ лӘ…мӢңм Ғмқё мӢңмһ‘ мӢ нҳёлҘј л°ӣм•ҳмқҢмқ„ мқҳлҜён•ңлӢӨ. мқҙ eventлҠ” мқјл°ҳм ҒмңјлЎң лӘ…мӢңм Ғмқё м •м§Җ(stop) нӣ„м—җ, мһ¬мӢңмһ‘н•ҳкё° мң„н•ҙм„ң мӮ¬мҡ©лҗҳм§Җл§Ң, мһҗлҸҷмӢңмһ‘(autostart)лЎң м„Өм •лҗҳм§Җ м•ҠмқҖ м»ҙнҸ¬л„ҢнҠёлҘј мӢңмһ‘н•ҳкё° мң„н•ҙм„ңлҸ„ мӮ¬мҡ©лҗңлӢӨ
ContextStoppedEventApplicationContextк°Җ м •м§Җн•  л•Ң л°ңмғқн•ңлӢӨ(м •м§Җн•ҳкё° мң„н•ҙм„ң ConfigurableApplicationContext interfaceмқҳ stop() л©”мҶҢл“ңлҘј мӮ¬мҡ©н•ңлӢӨ). вҖңм •м§ҖлҗЁ(Stopped)вҖңлһҖ лӢЁм–ҙлҠ”, лӘЁл“  Lifecycle beanмқҙ лӘ…мӢңм Ғмқё м •м§Җ мӢ нҳёлҘј л°ӣм•ҳмқҢмқ„ мқҳлҜён•ңлӢӨ. м •м§Җлҗң contextлҠ” start() нҳём¶ңмқ„ нҶөн•ҙ мһ¬мӢңмһ‘лҗ  мҲҳ мһҲлӢӨ.
ContextClosedEventApplicationContextк°Җ лӢ«нҳ”мқ„ л•Ң л°ңмғқн•ңлӢӨ(лӢ«кё° мң„н•ҙм„ң ConfigurableApplicationContext interfaceмқҳ close() л©”мҶҢл“ңлҘј мӮ¬мҡ©н•ңлӢӨ). вҖңлӢ«нһҳ(Closed)вҖңлһҖ лӢЁм–ҙлҠ”, лӘЁл“  singleton beanмқҙ нҢҢкҙҙлҗҳм—ҲмқҢмқ„ мқҳлҜён•ңлӢӨ. лӢ«нһҢ contextлҠ” мғқлӘ…мЈјкё°мқҳ лҒқм—җ лҸ„лӢ¬н•ң кІғмңјлЎң, refresh лҗҳкұ°лӮҳ мһ¬мӢңмһ‘лҗ  мҲҳ м—ҶлӢӨ.
RequestHandledEventмӣ№м—җ нҠ№нҷ”лҗң eventлЎңм„ң, HTTP requestк°Җ мІҳлҰ¬лҗҳм—ҲмқҢмқ„ м•ҢлҰ°лӢӨ(requestк°Җ мў…лЈҢлҗң нӣ„м—җ л°ңмҶЎлҗңлӢӨ). Springмқҳ DispatcherServletлҘј мӮ¬мҡ©н•ҳлҠ” мӣ№ м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқё кІҪмҡ°м—җл§Ң мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ.

мғҲлЎңмҡҙ eventлҘј кө¬нҳ„н•ҳлҠ” кІғлҸ„ м–ҙл өм§Җ м•ҠлӢӨ. ApplicationContext interfaceлҘј кө¬нҳ„н•ң мғҲлЎңмҡҙ event к°қмІҙлҘј ApplicationContextмқҳ publishEvent() л©”мҶҢл“ңлҘј нҶөн•ҙ л°ңн–үн•ҳл©ҙ лҗңлӢӨ. publishEvent() л©”мҶҢл“ңлҠ” лӘЁл“  listenerк°Җ event мІҳлҰ¬лҘј л§Ҳм№ л•Ңк№Ңм§Җ block мғҒнғңлЎң мһҲкІҢ лҗңлӢӨ. кІҢлӢӨк°Җ transaction contextк°Җ к°ҖлҠҘн•ҳлӢӨл©ҙ, listenerк°Җ eventлҘј л°ӣм•ҳмқ„ л•Ң, л°ңн–үлӘЁл“Ҳмқҳ transaction context лӮҙм—җм„ң eventлҘј мІҳлҰ¬н•ңлӢӨ.

   private List blackList;
   private ApplicationContext ctx;

   public void setBlackList(List blackList) {
       this.blackList = blackList;

       if (blackList.contains(address)) {
           BlackListEvent event = new BlackListEvent(address, text);
           ctx.publishEvent(event);
           return;
       }

   }
}

   }

       if (event instanceof BlackListEvent) {
           // notify appropriate person...
       }
   }
}

мӣ№ м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқ„ мң„н•ң нҺёлҰ¬н•ң ApplicationContext к°қмІҙнҷ”(Convenient ApplicationContext instantiation for web applications)

BeanFactoryк°Җ н”„лЎңк·ёлһЁм ҒмңјлЎң мғқм„ұлҗҳлҠ” кІғкіј л°ҳлҢҖлЎң, ApplicationContext к°қмІҙлҠ” ContextLoader л“ұмқ„ мӮ¬мҡ©н•ҳм—¬ м„ м–ём ҒмңјлЎң мғқм„ұлҗ  мҲҳ мһҲлӢӨ. л¬јлЎ  ApplicationContext к°қмІҙ м—ӯмӢң н”„лЎңк·ёлһЁм ҒмңјлЎң мғқм„ұн•  мҲҳ мһҲлӢӨ.
ContextLoaderм—җлҠ” ContextLoaderListenerмҷҖ ContextLoaderServletк°Җ мһҲлӢӨ. л‘ҳ лӢӨ кё°лҠҘм ҒмңјлЎңлҠ” к°ҷм§Җл§Ң, listener лІ„м „мқҖ Servlet 2.3 м»Ён…Ңмқҙл„Ҳм—җм„ңлҠ” мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ.

л”°лқјм„ң servlet context listnerк°Җ Spring ApplicationContextлҘј мҙҲкё°нҷ”н•  мөңм Ғмқҳ мһҘмҶҢмқҙлӢӨ.

ListenerлҠ” ‘contextConfigLocation’ нҢҢлқјлҜён„°лҘј кІҖмӮ¬н•ңлӢӨ. л§Ңм•Ҫ мЎҙмһ¬н•ҳм§Җ м•Ҡмңјл©ҙ кё°ліём ҒмңјлЎң /WEB-INF/applicationContext.xmlлҘј мӮ¬мҡ©н•  кІғмқҙлӢӨ.
л§Ңм•Ҫ ‘contextConfigLocation’ нҢҢлқјлҜён„° к°’мқҙ мЎҙмһ¬н•  кІҪмҡ°, лҜёлҰ¬ м •н•ҙлҶ“мқҖ кө¬л¶„мһҗ(comma(’,’), semicolon(’;’), кіөл°ұл¬ёмһҗ(whitespace))лҘј мӮ¬мҡ©н•ҳм—¬ нҢҢлқјлҜён„° л¬ёмһҗм—ҙмқ„ л¶„лҰ¬н•ң нӣ„, application contextлҘј м°ҫмқ„ кІғмқҙлӢӨ.
Ant-style path нҢЁн„ҙмқҙ м§ҖмӣҗлҗңлӢӨ. мҳҲлҘј л“Өм–ҙ /WEB-INF/*Context.xml’мқҖ вҖңWEB-INFвҖқ л””л үнҶ лҰ¬м—җ мЎҙмһ¬н•ҳлҠ” вҖңContext.xmlвҖқлЎң лҒқлӮҳлҠ” мқҙлҰ„мқ„ к°Җм§„ лӘЁл“  нҢҢмқјмқ„ мқҳлҜён•ҳкі , \
/WEB-INF/**/*Context.xmlмқҖ вҖңWEB-INFвҖқ л””л үнҶ лҰ¬ л°Ҹ н•ҳмң„ л””л үнҶ лҰ¬м—җ мЎҙмһ¬н•ҳлҠ” вҖңContext.xmlвҖқлЎң лҒқлӮҳлҠ” мқҙлҰ„мқ„ к°Җм§„ лӘЁл“  нҢҢмқјмқ„ мқҳлҜён•ңлӢӨ.
